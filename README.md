# Portable Electrocardiogram Device

Spanish version of the README: [README.es.md](./README.es.md)

This repository documents the design and development of a portable ECG system, from theory and signal processing to embedded communication, storage, hardware, and physical design.

## Project Overview

The goal of this project is to organize all the knowledge and work needed to build a portable electrocardiogram device in a clear, modular way.
