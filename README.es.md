# Dispositivo portátil de electrocardiograma

Este repositorio documenta el diseño y desarrollo de un sistema ECG portátil, desde la teoría y el procesamiento de señal hasta la comunicación embebida, el almacenamiento, el hardware y el diseño físico.

## Resumen del proyecto

El objetivo de este proyecto es organizar de forma clara y modular todo el conocimiento y el trabajo necesario para construir un dispositivo portátil de electrocardiograma.
